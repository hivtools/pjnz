## Installation

Please install from our
[r-universe](https://hivtools.r-universe.dev/builds):

``` r
install.packages(
  "pjnz",
  repos = c("https://hivtools.r-universe.dev", "https://cloud.r-project.org"))
```

You can install the development version of pjnz from [GitHub](https://github.com/) with:

``` r
# install.packages("remotes")
remotes::install_github("hivtools/pjnz")
```
